# AIspringFES 2026 NEWS管理ツール 利用マニュアル

本ドキュメントでは、ホームページの「NEWS」セクションを更新するための管理ツールの導入方法と操作手順について解説します。

---

## 1. 事前準備

ツールの実行には **Node.js** がインストールされている必要があります。

1.  **Node.jsの確認**: ターミナル（コマンドプロンプトやPowerShell）で以下を実行し、バージョンが表示されればOKです。
    ```bash
    node -v
    ```
    ※インストールされていない場合は [Node.js公式サイト](https://nodejs.org/) から推奨版をインストールしてください。

2.  **依存ライブラリのインストール**: プロジェクトのルートディレクトリ（`server.js` がある場所）で以下を実行します。
    ```bash
    npm install
    ```

---

## 2. ツールの起動方法

1.  ターミナルで以下のコマンドを実行します。
    ```bash
    node server.js
    ```
2.  実行後、ブラウザで以下のURLにアクセスしてください。
    *   **管理画面**: [http://localhost:3000/admin/index.html](http://localhost:3000/admin/index.html)
    *   **本番確認用**: [http://localhost:3000/index.html](http://localhost:3000/index.html)

---

## 3. 管理画面の操作説明

### 3.1 画面の構成
管理画面は、左側の「記事一覧」と右側の「編集エリア」に分かれています。

![管理画面全体](file:///C:/Users/tamor/.gemini/antigravity/brain/4a5f16b1-7e68-46d9-b9a5-84865fe99a07/admin_panel_full_1772800106226.png)

### 3.2 記事の追加・編集
*   **新規作成**: 左上の「+ 新規追加」ボタンを押します。
*   **編集**: 一覧から対象の記事をクリックすると、右側のエディタに内容が表示されます。

![編集画面](file:///C:/Users/tamor/.gemini/antigravity/brain/4a5f16b1-7e68-46d9-b9a5-84865fe99a07/admin_editor_item_01_1772800164102.png)

*   **詳細内容**: テキスト内のURLは自動的にクリック可能なリンクに変換されます。改行もそのまま反映されます。

### 3.3 記事の表示順序の変更
記事一覧の右側にある「▲」「▼」ボタンで、記事の並び順を自由に変更できます。変更は即座に保存されます。

### 3.4 プレビュー機能
「プレビュー」ボタンを押すと、実際のサイトでの表示イメージを確認できます。

![プレビュー画面](file:///C:/Users/tamor/.gemini/antigravity/brain/4a5f16b1-7e68-46d9-b9a5-84865fe99a07/admin_preview_modal_1772800175765.png)

## 4. GitHubへの反映（本番公開）

ローカルで行った変更（記事の追加、編集、並び替えなど）は、**GitHubに反映させるまで外部には公開されません。**

1.  管理画面右上の **「GitHubに反映 (本番公開)」** ボタンをクリックします。
2.  確認ダイアログが出るので「OK」を押します。
3.  ステータスが「完了しました」になれば、世界中のユーザーに最新のNEWSが公開されます。

![公開ボタン](file:///C:/Users/tamor/.gemini/antigravity/brain/4a5f16b1-7e68-46d9-b9a5-84865fe99a07/admin_deploy_button_1772800189492.png)

## 5. 注意事項

*   **サーバーの停止**: 作業が終わったら、ターミナルで `Ctrl + C` を押すとサーバーを停止できます。
*   **ローカル専用**: この管理ツールは開発者のPC内でのみ動作します。外部から `http://localhost:3000` にアクセスすることはできません。
